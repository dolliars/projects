Be Kind Bot

Inspired by a conversation between Shane Parrish and Adam Grant on The Knowledge Project podcast, I decided to write a twitter bot that asked the same question on a daily basis. One issue that came up was that the Twitter API throws an error when a duplicate status is detected. To circumvent the issue, I added the ordinal date as the first part of the tweet. I also created and launched an aws ec2 instance to serve the code.

tech : aws ec2, twitter api, javascript, git
